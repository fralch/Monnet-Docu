# Welcome!

## Welcome to Monnet Payout Doc!

The purpose of this documentation is to guide the developer on how to integrate with *Monnet Payments*, describing the features and methods to be used, listing information to be sent and received as well as providing examples.

The integration mechanism with *Monnet Payments* is simple, so only intermediate knowledge in any programming language, HTTP/HTTPS requests, and JSON file manipulation is required to successfully deploy the *Monnet Payout* solution.

In this manual, you will find a reference to all operations available on the API REST of the Monnet API
