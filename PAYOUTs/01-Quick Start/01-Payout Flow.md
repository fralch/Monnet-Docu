# Payout Flow

<figure><img src="https://1642515047-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMvYrq8H0R42hedo5qReG%2Fuploads%2FxecxGrF1vYaGrvcEmBT3%2Fpayout%20flow.png?alt=media&#x26;token=7b30be97-2abd-4276-a3f2-abe845520abe" alt=""><figcaption></figcaption></figure>

1. The customer confirms the payout on the merchant website.
2. The payout request travels to the merchant's server.
3. The merchant sends the payout request to Monnet.
4. Monnet sends the payout to the bank.
5. The user receives the amount on him/her bank account.
6. The bank sends the result to Monnet.
7. Monnet sends the result into a notification to the merchant.&#x20;
